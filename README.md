# snake-sweeper

🐍💣 **Snake-Sweeper** 是一款結合 **貪吃蛇（Snake）** 與 **踩地雷（Minesweeper）** 的創新遊戲，讓你在經典的貪吃蛇玩法中，透過數字提示策略性地遊玩！

## 🎮 遊戲玩法

- 控制貪吃蛇移動，吃掉隱藏的水果（地雷）。
- 每吃一顆水果，就會在空白處隨機生成新的水果。
- 蛇的每個身體部分都會顯示其周圍 8 格內的水果數量，類似踩地雷的數字提示。
- **遊戲結束條件**：
  - 撞到牆壁
  - 撞到自己的身體

## 🛠️ 技術棧

- **HTML5 Canvas** - 遊戲畫面繪製
- **JavaScript (Vanilla JS)** - 遊戲邏輯與事件處理
- **CSS** - 簡單樣式設計

| 按鍵  | 方向  |
|------|------|
| ⬆️ 上鍵 | 向上移動 |

| ⬅️ 左鍵 | 向左移動 |
| ➡️ 右鍵 | 向右移動 |
